# Smart Grid AI Project

## Goal
Forecast-aware Reinforcement Learning for intelligent microgrid energy management.

## What this project explores
- Time-series forecasting for energy demand and solar generation
- Reinforcement learning for energy optimization
- Smart grid simulation

## Structure
- 01_python: Python practice
- 02_algorithms: DSA problems
- 03_time_series: forecasting models
- 04_reinforcement_learning: RL experiments
- 05_smart_grid: system integration
- 06_project_notes: explanations and design decisions
- research_notes: paper summaries
- datasets: sample datasets

## Tech Stack
- Python
- PyTorch
- NumPy / Pandas
- OpenAI Gym (future)
